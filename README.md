# 블로그 프로젝트

이 프로젝트는 **멀리 떨어져 있는 팀원들 간의 원활한 협업을 돕는 웹 애플리케이션**입니다. <br/>
공식적인 블로그 플랫폼 없이도 **간단한 자기소개 및 글 게시가 가능한 웹사이트**기능이 포함되어 있습니다.<br/>
일인 프로젝트로 디자인부터 퍼블리싱, 정적 웹사이트 배포까지 진행하였습니다.
